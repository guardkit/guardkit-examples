# kartlog

## Project Overview

This is a JavaScript project using Svelte 5.35.5, Firebase 10.14.1 (Auth, Firestore), Vite 7.0.4, SMUI (Svelte Material UI) 8.0.3, svelte-spa-router 4.0.1.
Architecture: Client-Side MVC with Backend-as-a-Service (Firebase)

## Quick Start

```bash
# Install dependencies
npm install

# Run tests
pytest tests/ -v

# Start development
# See project documentation
```
