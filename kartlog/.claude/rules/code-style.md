# Code Style Guide

## Language: JavaScript

### Naming Conventions

- Functions/variables: camelCase
- Classes: PascalCase
- Constants: UPPER_CASE

### Formatting

- Use Prettier
- Max line length: 100 characters
- Use semicolons

### Best Practices

- Use const by default
- Avoid var
- Use arrow functions
